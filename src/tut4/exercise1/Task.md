# Task Description

Implement the following methods under [Main.java](Main.java):

```java
class Main {
    void printIntegers(int[] array) {
        // TODO: Implement me
    }
    
    void printMid(int[] array) {
        // TODO: Implement me
    }
}
```

The `printIntegers` method should:
- Take in `integer array`
- Print out the integers as single line
- Print `0` if array empty or `null`

The `printMid` method should:
- Take in `integer array`
- Print out the value between `max` and `min` values of the array
- Print `0` if array empty or `null`
- Example:
  - `{ 10, -12, 14 }` -> `max=14`, `min=-12`, `result=1`
